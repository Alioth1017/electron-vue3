# Electron_Template
electron & vue3 

## 安装依赖
运行命令
```shell
npm install
```
### 安装中的网络问题
- `electron`:
```shell
npm config set electron_mirror https://npm.taobao.org/mirrors/electron/
```

## 运行调试
运行命令
```shell
npm run dev:all
```
